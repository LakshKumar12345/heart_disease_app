## 🧠 Overview
This project is an AI-powered **Heart Disease Prediction Web Application** built using Flask and Machine Learning.

It predicts the **risk of heart disease** based on medical inputs and generates a **detailed AI medical report (PDF download supported).**

---

## ⚙️ Features

✔ AI-based heart disease prediction  
✔ Risk percentage calculation  
✔ Medical explanation (why prediction happened)  
✔ PDF report generation & download  
✔ Clean Flask web interface  
✔ Real-time analysis system  
✔ Deployed on Railway  

---

## 🫀 Input Features

The system uses the following medical parameters:

- Age  
- Gender  
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol Level  
- Fasting Blood Sugar  
- Rest ECG Results  
- Maximum Heart Rate  
- Exercise Induced Angina  
- ST Depression (Oldpeak)  
- Slope  
- Number of Major Vessels (ca)  
- Thalassemia  

---

## 📊 Output Categories

🟢 Low Risk  
🟡 Moderate Risk  
🔴 High Risk  

Each prediction includes:
- Risk Percentage (%)  
- Medical reasoning  
- Doctor recommendation
